# Smart Itinerary Planner (Planify)

A travel planning web app that helps users discover destinations and find nearby accommodations — filtered by price, rating, and proximity. Built as my BCA mini project.

`PHP` `MySQL` `HTML` `CSS` `JavaScript` `XAMPP`

---

## Technologies Used
- HTML, CSS, JavaScript (frontend)
- PHP (backend)
- MySQL (database)
- XAMPP (local server environment)

## Features
- User registration and login
- Browse destinations by category (beach, hill station, city, etc.)
- View destination details — name, location, category, description
- Explore nearby accommodations filtered by price, star rating, and proximity
- Save personalized plans under "My Plans"
- Admin panel — manage users, destinations, and accommodations (CRUD)


## Potential Improvements

- Cab/transport booking for local and intercity travel

- Real-time weather, traffic, and event updates

- Mobile app / responsive design
- Wishlist / save-for-later feature

## How to Run the Project
1. Install [XAMPP](https://www.apachefriends.org/)
2. Clone this repository into your `htdocs` folder
3. Start Apache and MySQL from the XAMPP control panel
4. Import the database file into phpMyAdmin
5. Open `http://localhost/miniproject/planify` in your browser
